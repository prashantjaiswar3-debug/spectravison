# **App Name**: CCTV Manager

## Core Features:

- Add Camera: Add new CCTV camera entries with details such as camera name, IP address, location, and installation date.
- View Camera List: Display a comprehensive list of all CCTV cameras with essential information in a sortable table.
- Update Camera: Update the status (active/inactive) and location of each CCTV camera.
- Search & Filter Cameras: Allow users to search and filter cameras by name, location, or status using an intuitive interface.
- Generate Camera Report: Use an AI tool to automatically compile reports that analyze existing cameras and their status to highlight those requiring attention, upgrades, or with outdated firmware.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) to convey professionalism and reliability.
- Background color: Desaturated dark blue (#242B34) to create a clean and modern dark theme.
- Accent color: Silver (#B4B4B4) for interactive elements, providing a sleek, high-tech feel.
- Body and headline font: 'Inter', a sans-serif font for a modern, machined, objective, neutral look.
- Use a set of simple, consistent icons related to camera management.
- A clean, organized layout for camera information and management functions.
- Subtle animations to highlight important interactions.
